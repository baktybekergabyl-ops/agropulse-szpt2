# АгроПульс СЗПТ

Публичная версия одностраничного дэшборда по социально значимым продовольственным товарам.

Сайт публикуется через GitHub Pages из ветки `main`, папка `/`.

В репозиторий не нужно добавлять:

- токен Telegram-бота;
- базу запасов `stocks.sqlite3`;
- исходные файлы областей;
- большие архивы БНС/Талдау.

Публичные файлы сайта:

- `index.html`
- `styles.css`
- `app.js`
- `data/bns.json`
